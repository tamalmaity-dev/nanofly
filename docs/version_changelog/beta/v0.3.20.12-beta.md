# NanoFly v0.3.20.12-beta — File Manager Media Previews & Expandable Domain DNS Diagnostics

**Release Date:** 2026-05-30  
**Type:** Feature & Enhancement Update  
**Urgency:** High  

---

## Summary

This release introduces rich, native media streaming and file viewer previews (Audio, Video, Images, PDFs) directly inside the NanoFly File Manager, powered by a new secure raw file streaming endpoint. Additionally, it upgrades the Custom Domains management interface to display real-time background-verified DNS routing diagnostics, IP mismatches, and step-by-step setup guides to troubleshoot failed domain mappings.

## Key Changes

### 1. Native Media Players & Viewers in File Manager
- **Dynamic File Type Parsing**: Added client-side detection for media file extensions:
  - **Audio**: `.mp3`, `.wav`, `.ogg`, `.m4a`, `.aac`, `.flac`
  - **Video**: `.mp4`, `.webm`, `.mov`, `.mkv`, `.avi`
  - **Images**: `.png`, `.jpg`, `.jpeg`, `.gif`, `.svg`, `.webp`, `.bmp`, `.ico`
  - **PDFs**: `.pdf`
- **Premium Media UIs**:
  - **Audio Player**: Renders a spinning vinyl disc animation when playing, a live active-pulse visualizer bar widget, and a modern custom player block.
  - **Video Player**: Custom viewport with styled dark-gradient background and player control bar.
  - **PDF Reader**: Direct iframe embedding for zero-config document viewing.
  - **Image Viewer**: Centered responsive light-box.
- **Secure File Streaming Endpoint**: Added `GET /api/v1/files/raw` which streams files directly using standard Go standard library `http.ServeFile` under the JWT query-parameter token verification handler.

### 2. Custom Domain Diagnostics & DNS Setup Guides
- **Premium Expandable Cards Layout**: Replaced the legacy table layout with interactive domain status cards.
- **Background Autochecking**: Runs dynamic verification checks in parallel asynchronously when navigating to the Domains page.
- **Detailed Routing Status**:
  - Displays target services, redirection rules, and project affiliations.
  - Highlights precise errors (e.g. resolved IPs vs server expected IPs) within color-coded status boxes (green for verified, red for warning).
- **DNS Troubleshooting Step-by-Step**: Explains exactly how to configure A-records on domain registrars and advises disabling Cloudflare's proxy (Grey Cloud) to allow Traefik Let's Encrypt certificate acquisition.
