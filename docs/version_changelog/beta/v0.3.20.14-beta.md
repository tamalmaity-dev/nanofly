# NanoFly v0.3.20.14-beta — Borderless File Manager Overhaul, Rename Actions & Interactive Image Editor

**Release Date:** 2026-05-30  
**Type:** Feature, Theme Support & Visual Redesign Update  
**Urgency:** High  

---

## Summary

This release introduces a major visual overhaul and feature expansion for the NanoFly File Manager. The interface now features borderless panels separated by subtle divider rules in a clean developer IDE column layout. Hardcoded dark colors are fully replaced with standard CSS theme variables to fix light-mode compatibility. Additionally, we added Drag-and-Drop uploads, folder-first sorting, folder/file renaming, support for a wider range of media formats, and an interactive Image Editor that supports scroll wheel zooming, click-drag panning, and realtime style adjustment filters.

## Key Changes

### 1. Theme Compatibility (Light/Dark Mode Fix)
- **Responsive Colors**: Replaced all hardcoded background and text values in the file manager with theme CSS variables (`var(--bg-base)`, `var(--bg-surface)`, `var(--bg-elevated)`, `var(--border)`, `var(--text-primary)`, etc.). The page now matches light and dark modes.

### 2. Seamless Borderless columns Divider Layout
- **IDE Column Layout**: Stripped individual container borders, box shadows, and margins to build a unified column system divided only by single vertical divider rules (`1px solid var(--border)`).
- **High-Density Sidebar**: Restyled drives and storage mounts to align with file explorer lists.

### 3. Drag-and-Drop File Uploads
- **Drop Zone Overlay**: Implemented HTML5 drag-and-drop support. Dragging files over the explorer list displays a drop overlay zone to upload multiple items in the background.

### 4. Folder-First Sorting Controls
- **Metrics Dropdown**: Sort folder lists dynamically by **Name**, **Size**, or **Last Modified Date** (Ascending/Descending).
- **Priority Directories**: Directory items are automatically prioritized and kept at the top of the sorted lists.

### 5. Files/Folders Renaming (API & UI Modal)
- **Rename Endpoint**: Registered `/api/v1/files/rename` on the Go backend using standard `os.Rename`.
- **Rename Dialog**: Added inline Rename row buttons that trigger a clean prompt input modal to submit rename requests.

### 6. Interactive Image Editor
- **Mouse Wheel Zoom**: Scroll over the canvas to zoom in/out dynamically.
- **Drag Panning**: Click and drag zoomed images in any direction.
- **Visual Filters**: Adjust image brightness, contrast, grayscale, and sepia values using flat sliders, alongside an Invert checkbox and 90-degree Rotation triggers.

### 7. Expanded Media Formats
- **Wide Extensions Support**: Enabled native playback recognition for a broader range of media:
  - **Images**: `.heic`, `.avif`, `.webp`, `.tiff`
  - **Audio**: `.m4a`, `.aac`, `.wma`
  - **Video**: `.mov`, `.mkv`, `.flv`, `.wmv`
