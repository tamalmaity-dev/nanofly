# NanoFly v0.3.20.13-beta — High-Performance Flat File Manager & ZIP/UNZIP

**Release Date:** 2026-05-30  
**Type:** Feature, Performance & UX Redesign Update  
**Urgency:** High  

---

## Summary

This release introduces comprehensive performance optimizations and visual redesigns for the NanoFly File Manager. Features include server-side ZIP archive zipping and unzipping, navigation path persistence across browser refreshes, big file click-interception warnings, and a syntax-highlighting bypass fallback to prevent browser page crashes. In accordance with design requests, the layout is redesigned to be flat and solid with high-contrast borders, completely removing glassmorphic transparency or heavy shadows.

## Key Changes

### 1. Server-Side ZIP/UNZIP File Compression & Decompression
- **Go Compression Handler**: Integrated standard Go package `"archive/zip"` in the backend to manage ZIP archiving.
- **Extraction API**: Added `/api/v1/files/unzip` to extract file archives directly on the server to target paths, including path traversal checks to secure against directory attacks.
- **Compression API**: Added `/api/v1/files/zip` to compress files or subfolders on the server into zip archives.
- **UI Integration**: Added inline Zip (for files/folders) and Unzip (for `.zip` archives) action buttons inside folder rows in both list and grid views.

### 2. URL Path Persistence
- **Query Parameter Sync**: Connected file manager state directly to the browser search bar with `?path=...` parameters, syncing directories dynamically.
- **State Save**: Saves directory states to `localStorage` as fallback. Pressing refresh or navigating back in history preserves folder location.

### 3. File Explorer Performance Optimizations
- **Large File Interception**: Prevent UI thread lockups by checking file sizes before loading.
- **Plaintext Fallback (> 500 KB)**: Automatically bypasses heavy syntax highlighting for files between 500 KB and 2 MB, loading them instead inside a high-speed `<textarea>` with mono-font.
- **Details Card Interception (> 2 MB & ZIPs)**: Directly intercepts ZIP files or files > 2 MB to render an overview card displaying file size, location, and quick actions: **Download File**, **Extract Archive**, or **Load as Text anyway**.
- **Solid Loaders**: Added flat panel dark loaders to show background fetches instead of presenting blank panels.

### 4. Flat Developer Design Overhaul
- **Flat Containers**: Replaced transparency, glassmorphism backdrop-filters, and drop-shadows with solid developer-centric panels (`#121420`).
- **High-Contrast Styling**: Utilizes crisp borders (`1px solid var(--border)`) and flat high-density layouts for folders and drives sidebar elements.
